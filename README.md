# QR 출입 관리 시스템

## 프로젝트 개요

QR-APP은 Android 기반의 QR 코드를 활용한 출입 관리 시스템입니다. 사무실, 학교, 상점 등 다양한 장소에서 효율적인 출입 관리와 통계 분석을 제공합니다.

## 주요 기능

### 사용자 관리
- 이름과 전화번호로 간단한 사용자 등록
- 개인별 고유 QR 코드 자동 생성
- 생성된 QR 코드를 SMS/MMS로 즉시 전송
- 등록된 사용자 목록 조회 및 관리

### QR 스캐너
- 실시간 카메라를 통한 QR 코드 스캔
- 입장/퇴장 상태 자동 토글 기능
- 오늘의 입장, 퇴장, 현재 체류자 수 실시간 표시
- 스캔 즉시 결과 처리 및 피드백

### Excel 리포트 생성
- 전체 출입 기록: 기간별 모든 출입 기록 추출
- 개인별 상세 기록: 특정 사용자의 상세 출입 내역
- 사용자 목록: 등록된 모든 사용자 정보 정리
- 일별 통계: 날짜별 출입 통계 및 근무시간 분석

### 파일 공유 시스템
- MMS 전송: Excel 파일을 직접 문자메시지로 전송
- 메신저 공유: 카카오톡, 텔레그램 등 메신저 앱 연동
- 일반 공유: Android 기본 공유 기능 지원
- 파일 크기 자동 체크: MMS 2MB 제한 사전 확인

## 기술 특징

- **100% Kotlin 개발**: 현대적이고 안전한 Android 개발
- **Jetpack Compose UI**: 선언적 UI로 일관된 사용자 경험
- **Material Design 3**: 최신 디자인 가이드라인 적용
- **Room Database**: 로컬 SQLite 데이터베이스로 빠른 데이터 처리
- **MVVM 아키텍처**: 유지보수 가능한 코드 구조
- **Apache POI Excel**: Android 최적화된 Excel 파일 생성
- **ZXing QR 라이브러리**: 고성능 QR 코드 생성 및 스캔

## 사용 대상

- **사무실 출입 관리**: 직원들의 출퇴근 시간 관리
- **학교 출석 관리**: 학생들의 출석 체크 및 통계
- **상점 고객 관리**: 단골 고객 방문 기록 및 분석
- **이벤트 참석 관리**: 행사 참가자 출입 통제
- **소규모 시설 관리**: 헬스장, 독서실 등 회원 출입 관리

## 주요 장점

- **간편한 설정**: 복잡한 설정 없이 즉시 사용 가능
- **오프라인 동작**: 인터넷 연결 없이도 완전 동작
- **실시간 통계**: 현재 상황을 한눈에 파악
- **다양한 리포트**: 필요에 따른 맞춤형 보고서 생성
- **파일 공유**: 생성된 리포트를 즉시 전송 가능
- **보안**: 로컬 데이터베이스로 개인정보 보호

QR-APP은 작은 규모부터 중간 규모의 조직에서 효율적인 출입 관리 솔루션을 찾는 분들에게 최적화된 앱입니다.
